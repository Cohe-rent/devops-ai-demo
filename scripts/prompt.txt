Generate a valid Terraform configuration for local development using the Docker provider from kreuzwerker. It should:
- Use the `kreuzwerker/docker` provider with version ~> 3.0.2
- Include a required_providers block inside `terraform {}`
- Include a `provider "docker"` block that uses the local Docker daemon
- Create an NGINX container listening on port 80
- Create a Python Flask app container listening on port 5000
- Use a Docker network so both containers can communicate
- The configuration must not use any cloud-specific backends like Azure or AWS
- The code should be complete and compatible with `terraform init` and `terraform apply`
